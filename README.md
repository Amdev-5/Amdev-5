### What I'm Building

**Multi-Agent Outreach System** — Two AI agents that collaborate to handle B2B sales:
- **Agent 1** researches prospects deeply (Apollo, Google Maps, web search), crafts personalized outreach, gets human approval before anything goes out
- **Agent 2** deploys as an intelligent meeting concierge on the prospect's side — answers questions, checks live availability, books the meeting, notifies the seller

The prospect doesn't get a cold email with a Calendly link. They get an AI concierge that already knows their company just raised Series B and they're scaling their sales team.

**Open Source Agent Framework Contributions:**
- Google Maps Platform integration (6 MCP tools — place search, details, directions, geocoding, distance matrix, nearby search)
- Apollo.io integration (people search, enrichment, org search)
- Cal.com integration (bookings, availability, event types, scheduling)
- X/Twitter integration (user lookup, timeline, search, lists)

Each integration: tool module + credential spec + full test coverage + live API validation.
